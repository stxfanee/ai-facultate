# Installation

## Normal users

1. Open the latest GitHub Release.
2. Download `Co-pilot Facultate Setup.exe`.
3. Run the installer.
4. Launch `Co-pilot Facultate`.
5. Create/select a profile and start chatting.

Normal users should not need PowerShell, BAT files, Ollama, ChromaDB, Streamlit or FastAPI.

## Portable app

If no installer is available, download `Co-pilot Facultate.exe` and run it directly.

## Server owner

Install the Python dependencies and Ollama on the desktop server PC. Then use the same `Co-pilot Facultate.exe` in Server Mode.
